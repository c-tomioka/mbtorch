MbTorch Web MLP Demo

Comparing 6 optimizer × activation combinations for a 2-layer MLP, entirely in the browser via WebAssembly.

Model: Linear(2,4) → activation → Linear(4,1) learns y = 2x1 + 3x2 from 3 data points.
Optimizers: SGD (lr=0.001) and Adam (lr=0.05). Activations: ReLU, tanh, sigmoid.
Each combination trains for 100 steps with identical initial weights.

This demo trains all 6 models from scratch in your browser.
Click the button to start training; once it finishes, the loss curves and results below are updated.
