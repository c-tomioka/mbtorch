MbTorch Web MLP Demo

Training a 2-layer MLP (Linear(2,4) → ReLU → Linear(4,1)) with SGD in the browser via WebAssembly.

The model learns y = 2*x1 + 3*x2 from 3 data points. Loss starts around ~137 and decreases to ~0.45 over 100 training steps — all computed inside the browser.
